# 模板目录结构

Ncms模板 遵循 “约定优于配置” 的原则，推荐的目录结构如下：

\app\view
├── cms
|  ├── demo
|  |  └── pc
|  |     ├── common
|  |     |  └── error403.html
|  |     └── index.html
|  └── www (站点目录)
|     ├── pc  (pc端模板目录)
|     |  ├── channel (栏目模板目录)
|     |  |  ├── channel.html
|     |  |  ├── channel_content.html
|     |  |  └── channel_resume.html
|     |  ├── common (反馈模板目录)
|     |  |  ├── error403.html
|     |  |  ├── error404.html
|     |  |  └── page_not_found.html
|     |  ├── content (内容模板目录)
|     |  |  └── content.html
|     |  ├── includes (公共模板目录)
|     |  |  ├── ad.html
|     |  |  ├── backtop.html
|     |  |  ├── footer.html
|     |  |  ├── head.html
|     |  |  ├── header.html
|     |  |  ├── meta.html
|     |  |  ├── nav.html
|     |  |  └── paging.html
|     |  ├── index.html (网站首页)
|     |  ├── layout.html (公共布局页)
|     |  └── search.html (搜索结果页)
|     └── wap (手机端模板目录)
|        ├── channel
|        |  └── channel.html
|        ├── common
|        |  ├── error404.html
|        |  ├── page_not_found.html
|        |  └── template_not_found.html
|        ├── content
|        |  └── content.html
|        ├── includes
|        |  ├── footer.html
|        |  ├── head.html
|        |  ├── header.html
|        |  └── meta.html
|        ├── index.html
|        ├── layout.html
|        └── search.html
├── cms_sys_defined
|  ├── common
|  |  ├── site_close.html
|  |  ├── site_not_found.html
|  |  └── template_not_found.html
|  ├── content

|  ├── includes
|  |  └── baidu.html
|  └── style_page
|     └── channel_1.html
└── tree.md

# 资源目录结构

Ncms资源 遵循 “约定优于配置” 的原则，推荐的目录结构如下：

\app\public\r      
└── cms
   ├── demo
   |  └── pc
   └── www
      ├── pc
      |  ├── css
      |  |  └── style.css
      |  ├── favicon.ico
      |  ├── img
      |  |  ├── bg.png
      |  └── js
      |     └── base.js
      └── wap
         └── css
            └── style.css
